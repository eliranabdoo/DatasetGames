Simply run either bike.py or imports.py for the analysis of the corresponding dataset.
===
PLOTS set to True would show the plots that were used in the EDA.
===
requirements file is provided, sklearn > 0.20 is strictly required, the code won't run without it.
Best to use it in venv as these are quite common libraries
===
custom_transformers and utils contain general functions that I used along the way,
they can be for other datasets as well.
===
